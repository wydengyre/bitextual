Bitextual: a two-minute tutorial

This tutorial teaches you to create a bilingual book from two epub files

Following the three steps below, we'll generate a bilingual, parallel, French-English version of Madame Bovary. By following these steps but substituting your own epub files, you can create bilingual versions of whatever book you want!

Step 1: download the epub files

We'll get both epubs from Project Gutenberg. Visit pages for the French version and the English version. On both pages, click to download both EPub files to your computer. Both EPUB and EPUB3 versions will work fine.

Step 2: enter the epub files

Head to bitextual.net. In the left-side, source pane, click the button to import the French epub you downloaded. On the right-side, import the English epub.

Note that both epubs contained a bit of "fluff" at the beginning: notes about the book, table of contents, etc. In our case, this won't mess with alignment, but if you're importing a different epub, you might want to manually delete this "fluff" so that both copies start at the exact same point, be it a prolog, or first chapter, etc. This especially applies when the translated version might have a lengthy, special introduction.

Step 3: Create the bilingual alignment

Click the big blue "generate" button. After a brief loading pause, you have a bilingual book! It will be similar to the example.

You can save your book using your browser's "save as" or "save page as", in its File menu.

Note that Bitextual is still very much an early beta product, with limited language support. It also has a tendency to screw up alignment on "liberally" translated text. For an example, see the alignment of Marianela. We're working on it!

Rejoice!
